# Computer Vision Lab

A collection of computer vision experiments built with Python, OpenCV and Streamlit.

## Features

- Image Upload
- Image Preview
- OpenCV Image Processing (Coming Soon)
- Edge Detection (Coming Soon)
- Histogram Analysis (Coming Soon)

## Technologies

- Python
- OpenCV
- Streamlit
- NumPy
- Pillow

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
